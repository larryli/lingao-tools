# 临高启明人物索引工具

## 安装

```shell
git clone https://github.com/larryli/lingao-tools.git lingao
cd lingao
composer install
git clone https://github.com/lingaoqiming/lingaoqiming.github.com.git book
```

## 增加索引

```shell
./lingao add book y001
./lingao add book y001 萧主任 子山
```

## 更新索引页

```shell
./lingao update book
```
